## Quick Reference

### Color Palette
- **Primary**: `#7A6A4D` (light) / `#B39668` (dark)
- **Accent**: `#A88D5E` (light) / `#C6A26B` (dark)
- **Background**: `#FCFCFC` (light) / `#272320` (dark)
- **Foreground**: `#2E2E2E` (light) / `#F5F5F4` (dark)

### Typography
- **Headings**: Geist, 700/600 weight
- **Body**: Geist, 16px, 400 weight
- **Monospace**: Geist Mono

### Spacing
- Base unit: 4px
- Common: 8px, 16px, 24px, 32px

### Components
- 58+ components available
- 6 button variants
- 15+ form controls
- Light & dark mode support
